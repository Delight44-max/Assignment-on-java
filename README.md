# Java Fundamentals & Algorithms — Semicolon Africa

A curated collection of Java programs covering core language concepts,
object-oriented design, and algorithmic problem-solving, written during
my training at Semicolon Africa in Lagos, Nigeria.

## Concepts Covered

- **OOP** — encapsulation, classes, methods, and object modelling
  (Bank, Health, StudentAge, Triangle, Geometry)
- **Control Flow** — conditionals, loops, and pattern generation
  (Evencheck, OddEven, Leapyear, Patterns)
- **Functions & Recursion** — pure functions, factorial, palindrome detection
  (Factorial, Pallindrome, Functionprime)
- **Algorithms** — sorting helpers, largest/smallest, prime checking, quadratic solver
  (Largest, LargestSmallest, Isprime, Quadratic)
- **Menu-Driven Programs** — simulated UI flows using switch and input handling
  (NokiaMenu, Selectmenu)
- **Mathematical Utilities** — converters, coin flip simulation, geometry calculators
  (Converter, Coinflip, Squarefunction)

## Tech

- **Language:** Java (100%)
- **Paradigm:** Object-Oriented Programming
- **Environment:** CLI / JDK
